# DesignReady — Landing Page

Marketing and info site for the [DesignReady Figma plugin](https://www.figma.com/community/plugin/1619643293052051000).

## Stack

- Astro 6 (static)
- Tailwind 4 via `@tailwindcss/vite`
- TypeScript strict
- Deployed via Cloudflare Pages

## Development

```bash
pnpm install
pnpm dev       # localhost:4321
pnpm build     # → dist/
pnpm preview   # serve dist/ locally
```
